# 🏦 Telegram Банковская система RP

Бот для Telegram-групп. Админы группы автоматически получают доступ к админ-командам.

## 🚀 Быстрый запуск локально

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Создать .env файл с токеном бота
echo "BOT_TOKEN=1234567890:ABCdef..." > .env

# 3. (Опционально) Если Telegram заблокирован — укажи прокси в .env
echo "BOT_PROXY=socks5://127.0.0.1:1080" >> .env

# 4. Запустить
python main.py
```

## 🤖 Создание бота

1. Открой @BotFather в Telegram
2. `/newbot` → название → `bank_rp_bot` (или любое)
3. Скопируй токен в `.env` файл: `BOT_TOKEN=твой_токен`
4. Отключи Privacy Mode в BotFather:
   - `/mybots` → выбери бота → **Bot Settings** → **Group Privacy** → **Disable**
   - Это нужно, чтобы бот видел список администраторов группы
5. Добавь бота в свою группу и дай ему права администратора (хотя бы "изменять информацию" — для чтения админов)

## ☁️ Деплой (бесплатный хостинг 24/7)

### 🔷 Fly.io (рекомендую) — 3 always-on инстанса бесплатно

```bash
# 1. Установить flyctl
# Windows (PowerShell как администратор):
# powershell -Command "iwr https://fly.io/install.ps1 -useb | iex"
# macOS / Linux:
# curl -L https://fly.io/install.sh | sh

# 2. Войти / зарегистрироваться
flyctl auth signup

# 3. Запустить деплой (в папке с проектом)
flyctl launch
# → На вопрос "Create copy?" → Yes
# → Название приложения (enter — автоматическое)
# → Регион — выбери ближайший (ams — Амстердам, fra — Франкфурт)
# → Deploy now? → No

# 4. Установить токен бота как секрет
flyctl secrets set BOT_TOKEN="твой_токен_бота"

# 5. Задеплоить
flyctl deploy

# 6. Проверить логи
flyctl logs
```

**Если надо перезапустить:**
```bash
flyctl apps restart
```

**Если надо обновить код:**
```bash
flyctl deploy
```

## 🇷🇺 Telegram заблокирован в России?

Если Telegram API недоступен, у бота не получится подключиться. Решения:

**Вариант 1 — Прокси в .env**
```env
BOT_PROXY=socks5://логин:пароль@сервер:1080
```

**Вариант 2 — Деплой на зарубежный сервер (рекомендую)**
Задеплой бота на **Fly.io** или **Koyeb** — сервер находится за пределами РФ, прокси не нужен. Полная инструкция выше.

**Вариант 3 — Использовать API-зеркало**
Замени в коде URL Telegram API на рабочее зеркало (если знаешь). Или используй VPN на своей машине.

## 🧠 Как это работает

- Бот добавляется в **группу**
- У бота должны быть права администратора (чтобы видеть список админов)
- Команды `/начислить`, `/списать`, `/одобрить_*`, `/отклонить_*` — только для админов группы
- Обычные участники могут переводить деньги, смотреть баланс/историю и отправлять заявки на кредит/вклад
- Заявки на кредит/вклад обрабатываются админами через `/одобрить_*` и `/отклонить_*`
- Все действия логируются в таблице `transactions`
